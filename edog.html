Website

A web app for buying and selling dogs. Made in Django (Python).

Features

Users

Users can sign up to create an account, sign in, view account details, and sign out.

Posts

Users can create posts for each dog they want to sell. Details can be filled out, and multiple photos can be used.

Pins

Users can pin posts they find, making it easy to locate later.

Technical Details

This is a web application made using the Django framework. Django uses Python for the back-end, and Templates for the front-end. Templates are primarily HTML, and can include CSS and Javascript as well.
